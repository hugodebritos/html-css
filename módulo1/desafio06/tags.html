Tags em HTML

Em HTML5, podemos inserir os elementos da lista a seguir. Passe o mouse sobre os termos para descobrir qual tag deverá utilizar em cada caso.

  • Títulos
  • Parágrafos
  • Links para outros endereços
  • Imagens
  • Código-fonte
  • Mexer no estilo da página
  • Para dar ênfase no texto forte (negrito)
  • Para dar ênfase no texto (itálico)
